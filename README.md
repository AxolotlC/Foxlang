# Fox programming language

Lightweight programming language written in C++ in under 4000 lines.

The goal of this language is to be C with rust-like syntax.

### Current Source Version: 1.0.2b1

### New features:
- C++ output
- Updated build system
- Char type
- C standard support
- Arrays
